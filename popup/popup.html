Azure DevOps Story Reviewer

Get AI-powered feedback on your user story in Azure Devops.

Click "Get Feedback" to receive feedback on your Story (opens a new window).

Valid Azure DevOps URLs

To use this extension, you must be on an Azure DevOps work item page. Valid URLs include:

  • dev.azure.com - e.g., https://dev.azure.com/yourorg/yourproject/_workitems/edit/123
  • visualstudio.com - e.g., https://yourorg.visualstudio.com/yourproject/_workitems/edit/123
